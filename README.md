## TZPipInstall
# Sobre:
<p>O TZPipInstall é um script que instala pacotes python e agiliza o processo "pip install"</p>

# Instalação:

```
$ apt update && apt upgrade -y
$ apt install git
$ git clone https://github.com/Odin-Hat/TZPipInstall
$ cd TZPipInstall
$ chmod +x TZPipInstall
$ ./TZPipInstall
```
